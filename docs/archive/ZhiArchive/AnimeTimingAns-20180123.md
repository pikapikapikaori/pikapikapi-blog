# 如何理解动画中的踩点，轨道，并用这个来分辨原画师？

> 本文首发于[知乎](https://www.zhihu.com/question/35917115/answer/302277374)\
> 发表日期：2018.01.14\
> 最后编辑于：{docsify-last-updated}

## 什么是踩点和轨道？

首先回答第一个问题：

> 如何理解动画中的踩点，轨道？

所谓的动画制作中的踩点、轨道，最初是通过 layout 上的轨目来表达的，当这个 layout 交给演出和作画监督检查，通过之后返还给原画师以进行正式的原画绘制工作时，原画师会绘制在原画上。在这里，踩点是指一个动作的速度快慢，而轨道指的是一个动作的路线。举个例子：比如说挥手这个动作，挥手的路线是从左至右还是从右至左，就是动作轨道（简称为轨道），而在挥手的路线上，什么时候手挥到哪里，花多久挥完，就是踩点。

## 怎么分辨原画师？

再回答第二个问题：

> 如何用动画中的踩点，轨道来分辨原画师？

在动画制作流程中，通常是由监督或各集分镜来制作分镜表，而原画师根据这个分镜表来制作 layout。一般来说一个角色做怎样的动作在分镜表中就已经有所描述，而并没有具体的轨目的描述，因而 layout 的轨目一般由原画师来绘制。但 layout 要通过监督的检查，此时监督通常也会对原画师的轨目提出一些意见，监督通过 layout 以后原画师才能将轨目具体为踩点和轨道，因而很大程度上踩点与轨道还是根据监督的意见来决定，很难从中分辨出原画师。

当然也不是说分辨不出。在获得一个动画的 STAFF 后，通常我们会得到参与动画制作的多位原画师，他们很难做到轨目绘制的习惯与水平相同。如果想简单分辨的话，可以通过观察绘制一个镜头时所用的电影胶卷格数（或称帧数）。多数原画师更偏向于一个镜头仅使用一格或两格（成为关键帧），而其他格由中割人员负责（即过渡帧），以确保画面的流畅。而更厉害一些的原画师更喜欢一个镜头使用三格或四格，从而避免使用中割（中间画）来填充过渡帧，防止因中割人员水平不足导致过渡帧的崩坏。虽然在最后会由监督进行画风的统一，但如果能获得一个镜头每一帧的图像，很明显就能分辨出哪些是由原画师负责的关键帧，哪些是由中割人员负责的过渡帧，由此来简单分辨原画师。

对于过去的动画还可以通过一个动作的时间精准度来分辨原画师。过去原画师是通过自己的时间感来踩点的，因而一个动作的踩点在不同原画师里都有些区别。而现在多使用计算机进行踩点测试，因而难以通过这个分辨原画师了。
